How to create and push the repository which is not available in git i.e online

NOTE: After we commit the changes we will push it to github for that the command is git push origin main where origin the variable holding link of the repository

To initialize new repository which is not available in github online we follow this steps

Step 1 - Create a new folder and name it.

Step 2 - Run this command git init this initializes the folder as new git repository

step 3 - Do some changes like add file

Step 4 - Run this command git status for the status and git add . to stage or structure the git.

Step 5 - Commit it with git commit "message" "description"

Step 6 - Create new repository in github

Step 7 - Run this command git remote add origin "url"

Step 8 - Run this command git pushgives error yelling git push --set-upstream origin master that it is asking us to push the correct branch and set the remote as upstream

Step 9 - Run this command git push --set-upstream origin master or git push -u origin master
